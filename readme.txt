== Description ==

This plugin allows to add text to the end of post/page, that we can hide/show.

== Installation ==

1. Upload the ss-hidden-text folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Adjust the plugin settings
